Nintendo Entertainment System

Released: July 15th, 1983

The NES is an 8-bit video game console, and Nintendo’s first game console released outside of Japan.

Popular Releases

  • Legend Of Zelda
  • Super Mario Bros.
  • Castlevania
  • Metal Gear

Specifications

  • GPU - Ricoh 2A03 8-bit processor (MOS Technology 6502 core)
  • Media - ROM cartridge ("Game Pak")
  • Inputs - 2 controller ports, 1 expansion port

Super Nintendo Entertainment System

Released: November 21st, 1990

The SNES is a 16-bit video game console, and was the best selling console of the 16-bit era.

Popular Releases

  • Super Mario World
  • Donkey Kong
  • Mega Man

Specifications

  • CPU - 16-bit 65816 (3.58MHz)
  • Colors - 32768 (256 on screen)
  • Resolution - 512x448 pixels.

Nintendo 64

Released: June 23, 1996

The Nintendo 64 was the first one ever to feature 3D graphics. It was also the first console to have an analog stick. This was one of the only older Nintendo consoles that you can still buy today for cheaper than release price.

Popular Releases

  • Pokemon Stadium
  • Super Mario 64
  • Super Smash Bros.
  • Golden Eye 007

Specifications

  • CPU - 64-bit Reality Coprocessor
  • Media - Nintendo 64 Game Pak
  • Memory - 4 MB Rambus RDRAM (8 MB with Expansion Pak)

GameCube

Released: November 5, 2001

The GameCube was Nintendo's first game console to use game discs rather than cartridges.

Popular Releases

  • Super Smash Bros. Melee
  • The Legend of Zelda: The Wind Waker
  • Animal Crossing
  • Resident Evil 4

Specifications

  • CPU - 32-bit IBM PowerPC 750CXe Gekko @ 485 MHz
  • Media - GameCube Game Disc
  • Memory - 24MB of 1T-SRAM @ 324MHz as system RAM, 3MB of embedded 1T-SRAM as video RAM, 16MB of DRAM as I/O buffer RAM

Wii

Released: November 19, 2006

The Wii was marketed towards people of all ages, bringing a new era of interactive gaming.

Popular Releases

  • Wii Sports
  • Mario Kart Wii
  • The Legend of Zelda: Twilight Princess
  • Just Dance

Specifications

  • CPU - IBM PowerPC Broadway @ 729 MHz
  • Media - Wii Optical Disc
  • Memory - 24 MB 1T-SRAM @ 324 MHz (2.7 GB/s) + 64 MB GDDR3 SDRAM

Wii U

Released: November 18, 2012

The Wii U was Nintendo’s first that supported HD graphics. It is unique in that the controller can become dependent and be used as the screen itself.

Popular Releases

  • Mario Kart 8
  • Super Mario 3D World
  • Splatoon
  • The Legend of Zelda: Breath of the Wild

Specifications

  • CPU - 1.24 GHz Tri-Core IBM PowerPC "Espresso"
  • Media - Physical and digital
  • Memory - 2 GB DDR3

Nintendo Switch

Released: March 3, 2017

The Nintendo Switch is a hybrid console that can be used as a handheld gaming system, stand alone monitor, or connected to a tv.

Popular Releases

  • Mario Kart 8 Deluxe
  • Animal Crossing: New Horizons
  • Pokémon Sword and Shield
  • Pokémon: Let's Go, Pikachu! and Let's Go, Eevee!

Specifications

  • CPU - ARM 4 Cortex-A57 cores @ 1.02 GHz
  • Media - Physical and Digital
  • Memory - 4 GB LPDDR4 @ 1331/1600 MHz
